# Telegram Bot для сжатия изображений

Telegram бот, разработанный на aiogram, который предоставляет возможность сжатия фотографий с использованием различных методов.

## Функциональность

Бот поддерживает три метода сжатия изображений:

1. **JPEG сжатие** - классическое сжатие с настройкой качества
2. **Вейвлет-сжатие** - современный метод на основе вейвлет-преобразования
3. **Квантовое сжатие** - процесс уменьшения количества различных цветов, используемых в изображении

## Как использовать

1. **Запустите бота** командой `/start`
2. **Выберите метод сжатия** из предложенных вариантов
3. **Отправьте фотографию** которую нужно сжать
4. **Получите результат** - сжатое изображение будет отправлено обратно

## Установка и запуск

### Требования

Убедитесь, что у вас установлен Python 3.7+

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Настройка бота
1. Создайте бота в Telegram через @BotFather
2. Получите API токен
3. Создайте файл .env в корне проекта и добавьте токен в формате
```env
BOT_TOKEN = ваш_токен_бота_здесь
```

## Важно
1. Файл должен называться именно **.env** (с точкой в начале)
2. Не добавляйте кавычки вокруг токена
3. Не допускайте лишних пробелов вокруг знака **=**

## Запуск
```bash
python bot.py
